# Android WebView — оболочка для Streamlit

1. Установите [Android Studio](https://developer.android.com/studio).
2. **File → Open** → выберите папку `android_webview`.
3. Отредактируйте `app/src/main/res/values/strings.xml` — укажите URL Streamlit Cloud.
4. **Build → Build Bundle(s) / APK(s) → Build APK(s)**.
5. APK: `app/build/outputs/apk/debug/app-debug.apk`.

Требуется Android SDK 34, JDK 17 (ставится с Android Studio).
